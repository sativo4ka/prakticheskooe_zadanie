Информационная система для автоматизации работы службы доставки. Проект включает в себя базу данных SQLite, графический интерфейс (GUI), консольный интерфейс (CLI), модули экспорта/импорта данных и систему логирования ошибок.

Стек технологий

* **Язык программирования:** Python 3.8+
* **База данных:** SQLite3
* **Графический интерфейс:** Tkinter
* **Консольный интерфейс:** Argparse
* **Тестирование:** Pytest

## Установка и запуск

1. **Установите необходимые зависимости:**
```bash

```



pip install -r requirements.txt

```

2. **Запуск приложения с графическим интерфейсом (GUI):**
   ```bash
   python main_gui.py

```

3. **Запуск консольных команд (CLI):**
* Просмотр аналитического отчёта по выручке:
```bash

```





python main_cli.py report

```
   * Экспорт данных в формат JSON:
     ```bash
     python main_cli.py export --file data/orders_backup.json

```

##  Тестирование

В проекте реализованы модульные тесты для проверки корректности работы с базой данных (CRUD-операции, расчет выручки и выборка топ-клиентов).

Для запуска тестов выполните команду в виртуальном окружении:

```bash
python -m pytest

```

##  Структура проекта

* `database.py` — модуль взаимодействия с БД (SQLite)
* `main_gui.py` — графический интерфейс пользователя
* `main_cli.py` — консольный интерфейс приложения
* `data_export.py` — функции импорта и экспорта JSON
* `logger_config.py` — централизованная настройка логирования
* `tests/` — папка с автоматическими тестами
